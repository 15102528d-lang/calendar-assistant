# Auto Calendar

Auto Calendar helps you extract schedule information from images, files, or chat records and export events into a system calendar friendly format.

## Main Features

- Upload screenshots, documents, or copied conversation text
- Parse and structure event information
- Edit event title, date, and time before export
- Export as ICS for importing into Apple Calendar, Google Calendar, and others

## Local Development

1. Install dependencies:

```bash
pnpm install
```

2. Start development:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm build
```

## Project Structure

- `client/`: frontend app
- `server/`: backend services and API
- `shared/`: shared types and utility modules
